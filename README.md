## Ejecución

Clonar el repositorio:

```text
git clone https://github.com/areelu/practica3.git
```
Compilar el archivo test2.c:

```text
mpicc main.c -o name
```
Correr el archivo compilado:

```text
mpirun -n 4 ./name 0 2 4 8
```

Este programa requiere tener instalado mpi.
